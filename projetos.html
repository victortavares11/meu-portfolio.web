Projeto Destaque

Portfólio Digital e Perfil Profissional

Contexto: Projeto desenvolvido para a cadeira de Introdução à Computação na CESAR School.

Descrição: Este projeto consistiu na criação de uma identidade digital profissional completa. O objetivo foi desenvolver um site de portfólio pessoal responsivo e estruturar perfis profissionais em plataformas como GitHub e LinkedIn.

Tecnologias utilizadas:

  • HTML5 (Estrutura Semântica)
  • CSS3 (Flexbox e Estilização)
  • JavaScript (Interatividade)
  • Git e GitHub (Versionamento)

Outras Atividades

Além do portfólio, estou constantemente estudando novas tecnologias e práticas de desenvolvimento web para aprimorar minhas habilidades em Front-end.

Desenvolvido por Victor Tavares - CESAR School
